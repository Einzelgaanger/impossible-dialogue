## Development

This is a React + TypeScript + Vite application with Tailwind CSS and shadcn/ui components.

**Run locally:**
```bash
npm install
npm run dev
```

**Build for production:**
```bash
npm run build
npm start
```

## Deployment

The application is deployed on Render and automatically builds from the main branch.
